Project Brief

Assessment Intelligence

Public-safe assessment system work: analytics, reproducible R processing, dashboarding, diagnostics, and decision-support reporting for mathematics programs.

Open Dashboard GitHub Repo

Overview

From assessment design to decision support

This project demonstrates how a department or school can move from assessment design to analysis and reporting without exposing real student records. The public analytics layer uses synthetic assessment artifacts and keeps the core data-generation foundation in synthetic-math-department.

Current Artifacts

  • Hosted synthetic assessment dashboard
  • R synthetic-data and modeling pipeline
  • Connection to the synthetic math department data foundation
  • Synthetic Canvas-style gradebook workflow
  • Privacy validation script
  • Dry-run leadership memo generator

Portal Notes

What this project proves

Static dashboard deployment

The dashboard runs on GitHub Pages with local JSON, browser-side aggregation, SVG charts, and no external JavaScript dependencies.

Public-safe synthesis

The assessment repo stages analytics and reporting workflows on public-safe synthetic outputs, while the separate synthetic math department repo owns the reusable generation and validation story.

Next milestone

Convert the gradebook synthesis and leadership memo artifacts into tighter reviewer-facing summaries as the reporting layer matures.

Safety Boundary

Synthetic by design

Public artifacts must not include real student names, emails, IDs, grades, rosters, LMS exports, submissions, or private school reporting artifacts. Canvas is treated as one possible adapter, not the project identity.
